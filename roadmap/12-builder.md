# Stage 1 — Builder (Age 12)

## Mission

Become someone who can turn ideas into reality.

Your goal this year is simple:

**Build. Build. Build.**

Forget about becoming a programming expert.

Instead, become someone who enjoys creating.

---

## Annual Goal

Complete at least 12 projects.

One project every month.

---

## Suggested Projects

- Snake
- Pong
- Flappy Bird
- Breakout
- Space Shooter
- Maze Generator
- Solar System
- Platformer
- Minecraft 2D
- Tower Defense
- Christmas Game
- Your Own Idea

---

## Engineering Habits

Learn how to:

- Use Git
- Write README files
- Commit code
- Debug patiently
- Ask good questions
- Finish projects

---

## Graduation Project

Build a game you are proud to share.
